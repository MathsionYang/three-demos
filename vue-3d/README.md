# 三维可视化

## 项目结构

- ***src***
    - assets ->fold 静态资源
    - components ->fold 组件
    - router ->fold 路由
    - store ->fold 状态管理
    - utils ->fold 函数
    - views ->fold 视图页面
    - App.vue ->file 根组件
    - main.ts -file 入口文件


## 项目命令

### 下载依赖

```js
npm install
```

### 开启服务

```js
npm run serve
```

### 打包

```js
npm run build
```

### 校验

```js
npm run lint
```
